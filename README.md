# POI采集核验应用官网

## 应用介绍
本应用为学生课程作业项目，专注于POI（兴趣点）数据的采集、上传与核验功能，支持用户通过微信登录完成身份验证，全程仅用于学习测试，不涉及任何商业用途、不对外提供公开服务，仅作为课程实践成果展示。

## 核心功能
1.  **地图选点采集**：支持用户通过地图定位，手动选择POI兴趣点，填写名称、坐标、类型等基础信息；
2.  **POI数据核验**：可导入已有POI数据，对比地图实际位置，验证数据的准确性和完整性；
3.  **数据存储测试**：支持采集数据本地存储（仅保存在用户设备，不上传服务器），用于课程实践中的数据处理测试。
